feat(auth): enable local password configuration for OAuth accounts

Allow setting and changing passwords for accounts managed by Google OAuth
as well as local accounts. Setting a local password on an OAuth-linked
account allows the user to optionally log in using email/password or
Google OAuth.

* Update getSSOSessionUser and /api/auth/me to fetch and return hasPassword status.
* Add a new /api/auth/change-password endpoint to handle password configuration.
* Add an Account Security panel and form to the central authenticated dashboard.
