## [0.1.2-alpha.1] - 2026-06-24

## New Features
- **Terms of Service & Privacy Policy Overlays**: Integrated modular legal overlay components.
- **Direct View Param Handling**: Supports direct navigation to terms or privacy views via URL query parameters (e.g. `?view=privacy` or `?view=terms`), ensuring OAuth compliance.
- **Brand Identity**: Added the logo directly onto the connection and dashboard views.
- **Release Integration**: Add the centralized `npm run release` script to support automated versioning, git tagging, changelog rolling, and GitHub release publishing.
